## Stack
React + Vite + Tailwind + Framer Motion
Fonts: Cormorant Garamond (display) + DM Sans (body)

## Colors
Background: #080808 | Accent: #C8FF00 | Text: #F5F5F0 | Muted: #555555

Inspired by ascendmarketing.xyz — cinematic, dark, editorial.
Full-screen sections. Massive Cormorant typography. Extreme negative space.
No rounded cards. Sharp edges. Hairline dividers. Zero clutter.
Text reveal via clip-path wipe. Scroll fade-ins with Framer Motion.
